# 打包说明

## 已完成的步骤

✅ 1. 创建程序图标（icon.ico）
✅ 2. 在程序中应用图标
✅ 3. 初始化Git仓库并推送到GitHub
✅ 4. 打包程序为exe文件

## 打包结果

打包完成后，可执行文件位于：
```
dist/歌词处理工具.exe
```

## 使用方法

### 方式一：直接运行exe
1. 进入 `dist` 目录
2. 双击 `歌词处理工具.exe` 即可运行
3. 该exe文件可以在任意Windows平台（Windows 7及以上）直接运行，无需安装Python

### 方式二：重新打包
如果需要重新打包，运行：
```bash
build.bat
```
或
```bash
pyinstaller build_exe.spec
```

## 文件说明

- `build_exe.spec` - PyInstaller打包配置文件
- `build.bat` - Windows批处理打包脚本
- `icon.ico` - 程序图标文件
- `dist/歌词处理工具.exe` - 打包后的可执行文件

## 注意事项

1. exe文件大小约为30-50MB（包含Python运行时和PyQt6库）
2. 首次运行可能需要几秒钟加载时间
3. 如果被杀毒软件误报，请添加信任
4. 确保目标Windows系统支持.NET Framework（通常Windows 7及以上都支持）
